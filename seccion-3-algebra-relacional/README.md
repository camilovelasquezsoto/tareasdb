# Sección 3: Álgebra Relacional

Esta sección agrupa los entregables teóricos y de traducción que vinculan las consultas estructuradas SQL con los lenguajes de consulta formales (Álgebra Relacional y Cálculo Relacional).

## Tareas y Ejercicios

### [01. Álgebra y Cálculo Relacional](./01-algebra-calculo-relacional/README.md)
*   **Contenido:** Expresión equivalente en álgebra relacional, cálculo relacional y SQL para resolver consultas sobre el esquema de ejemplo de Marineros (Sailors, Boats, Reserves).
*   **Enunciado guía:** [10-algebra-calculo-relacional](https://github.com/Awerito/db-ejemplos/tree/master/10-algebra-calculo-relacional)

### [02. Traducciones SQL y Álgebra](./02-traducciones/README.md)
*   **Contenido:** Documento `traducciones.md` con 10 ejercicios traducidos de SQL a Álgebra y viceversa.
*   **Enunciado guía:** [11-sql-algebra-traducciones](https://github.com/Awerito/db-ejemplos/tree/master/11-sql-algebra-traducciones)

### [03. Traducciones SQL y Álgebra (Set 2)](./03-traducciones-set-2/README.md)
*   **Contenido:** Un set avanzado de 40 ejercicios (20 traducciones SQL a Álgebra, y 20 de Álgebra a SQL) aplicados sobre 4 mini-esquemas de bases de datos.
*   **Enunciado guía:** *Set de ejercicios avanzados provistos en clase.*
