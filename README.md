# Tareas de Base de Datos

Repositorio único para las entregas del curso de Base de Datos.

## Estructura del repositorio

- **seccion-1-modelamiento/**: Tareas de modelo entidad-relación, modelo relacional y normalización.
- **seccion-2-sql/**: Scripts DDL, consultas CRUD, migraciones y triggers.
- **seccion-3-algebra-relacional/**: Consultas y traducciones en álgebra y cálculo relacional.
