# Tarea 04: Migraciones

Gestión de la evolución del esquema de la base de datos a través de migraciones numeradas e incrementales.

## Requisitos de Entrega
*   Una subcarpeta `migraciones/` con los archivos numerados del `001` al `009`.
*   Cada migración debe tener su script de subida (Up) y script de bajada o reversión (Down / Rollback).
*   Estado final y logs registrados en la tabla `schema_migrations`.
*   Enunciado guía: [13-migrations](https://github.com/Awerito/db-ejemplos/tree/master/13-migrations)

*Organiza tus archivos de migración dentro de la carpeta `migraciones/`.*
